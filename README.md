# Introduction to Online Feature Store in Snowflake

## Overview
Through this quickstart guide, you will explore the Snowflake Online Feature Store and learn how to build an end-to-end machine learning workflow. You will register entities and feature views, perform feature engineering, and use both online and offline stores for real-time inference to predict NYC taxi trip durations.

## Step-By-Step Guide
For prerequisites, setup, step-by-step guide and instructions, please refer to the [QuickStart Guide](https://www.snowflake.com/en/developers/guides/intro-to-online-feature-store-in-snowflake/).
